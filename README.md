# Git

Git is a tool that records the history of your project, so you can track changes, collaborate with others, and restore previous versions if needed.

## Why do we use Git?
- ✅ Tracks every change made to your files.
- ✅ Lets you go back to previous versions if something goes wrong.
- ✅ Allows multiple developers to work on the same project.
- ✅ Helps manage different features using branches.
- ✅ Works even without an internet connection (local repository).
